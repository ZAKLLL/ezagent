# 知识笔记库 📚

个人知识笔记的沉淀空间，按主题分类存放，用于积累行业认知、工作方法和个人成长记录。

## 📁 目录结构

```
00_Archive/           # 归档内容（过期资料、历史记录）
01_行业知识/           # 所在行业的知识、趋势、术语
02_工作方法/           # 项目管理、时间管理、汇报技巧等方法论
03_会议与沟通/         # 会议纪要模板、沟通技巧、谈判要点
04_职业发展/           # 职业规划、技能提升、绩效复盘
05_工具与效率/         # 常用办公工具、效率技巧
06_个人成长/           # 读书笔记、反思、习惯养成
```

每个分类下已放置一篇示例笔记，供你参考格式；替换为自己的内容即可，也可以随时新增分类。

## 使用说明

1. **按主题查找**：根据目录结构导航到对应分类
2. **搜索功能**：用文件搜索快速定位内容
3. **持续更新**：新笔记按此结构分类存放，分类不够用时可以直接新建

## 可选：用 Obsidian 管理

如果你习惯用 Obsidian 之类的双链笔记软件，可以直接把 `notes/` 目录作为 vault 打开，详见 `OBSIDIAN_NOTES.md`。这一步是可选的，普通 Markdown 编辑器也完全够用。
